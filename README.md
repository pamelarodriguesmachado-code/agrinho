# 🌾 Agro Forte, Futuro Sustentável — Projeto Agrinho

Este é um projeto de site institucional desenvolvido para o **Concurso Agrinho**, com o tema focado no equilíbrio entre a alta produtividade no campo e a preservação do meio ambiente. O objetivo é mostrar como a tecnologia e as boas práticas agrícolas podem caminhar juntas para construir um futuro sustentável.

## 🚀 Tecnologias Utilizadas

O projeto foi construído do zero, utilizando tecnologias web nativas para garantir leveza, rapidez e acessibilidade:
* **HTML5:** Estruturação semântica de todo o conteúdo do site.
* **CSS3:** Identidade visual responsiva, utilizando paleta de cores inspirada na natureza (tons de verde e terra) e efeitos visuais modernos.
* **JavaScript (ES6):** Interatividade para o menu de navegação e validação dinâmica do formulário de contato.

## 💡 Pilares Abordados no Site

O site destaca três práticas fundamentais para o agro sustentável:
1.  **Plantio Direto:** Proteção e conservação do solo.
2.  **Irrigação Inteligente:** Uso eficiente da água com o apoio de tecnologia.
3.  **Bioinsumos:** Defesa das lavouras de forma biológica e natural.
